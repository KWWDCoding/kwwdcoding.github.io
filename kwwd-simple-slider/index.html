🖼️ Simple Slider

A powerful image slider carousel plugin for WordPress. Create responsive carousels perfect for displaying products, Amazon affiliate links, photos, and more.

Everything You Need to Create Beautiful Sliders

Simple Slider gives you full control over how your image carousels are created, styled, and displayed.

🖼️

Multiple Sliders

Create unlimited sliders, each with its own unique set of images and settings. Perfect for different products or page sections.

⚙️

Global & Per-Slider Settings

Configure settings globally for consistency across all sliders, or customize individual sliders for specific needs.

🗨️

Image Captions

Add captions to each slide. Display titles, descriptions, or links - great for Amazon affiliate products or photo galleries.

🔗

Auto Trackable Shortlinks

If you have the URL Shortify plugin installed, a trackable shortlink is automatically created and applied to each slide for click tracking.

📊

CSV Bulk Upload

Upload multiple slides at once using CSV. Reference full image URLs from your media library - great for adding large batches of products.

📱

Responsive Design

Fully responsive sliders that work beautifully on any device. Display anywhere using shortcodes.

Quick Start Guide

Get started with Simple Slider in minutes.

1

Create a Slider

Go to Simple Slider in your WordPress admin and click "Add New" to create your first slider.

2

Add Slides

Upload images via the media library, then add them to your slider. Add captions and links as needed.

3

Configure Settings

Set global defaults or customize individual slider settings like transition speed and autoplay.

4

Display with Shortcode

Use [simple_slider id=X] to display your slider anywhere on your site.

Note: You need to create the slider first before accessing the bulk upload feature. Images in the CSV must be full URLs to images already uploaded to your media library.
